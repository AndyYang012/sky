# 光遇小工具中心 Sky Tools Hub

這是為《Sky 光遇》玩家製作的小工具網站，集中提供身高查詢與遊戲攻略，讓常用資訊更容易查找與使用。

網站：https://andyyang012.github.io/sky/

## 功能介紹

- **身高查詢**：上傳 Sky 角色 QR Code 截圖或貼上連結，查詢角色身高與可變化範圍。
- **小金人攻略**：依地圖整理攻略內容，方便玩家尋找小金人與翼能量。
- **中英切換**：主畫面與身高查詢頁支援繁體中文及 English。
- **Discord 社群連結**：提供玩家回報問題與交流使用。

## 頁面

- `index.html`：主畫面與工具入口。
- `height.html`：身高查詢工具。
- `golden-guide.html`：小金人攻略頁。
